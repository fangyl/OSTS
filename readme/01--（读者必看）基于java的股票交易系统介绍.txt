	随着现在互联网的出现，互联网借助其浏览量大，曝光率高的特点，可以在其平台建立相应的股票交易网站，
人们可以通过注册成为该网站会员，进行新闻浏览了解最新股票信息，然后进行相关的股票交易。这个项目分为
网站后台管理和前台管理系统。

基本功能介绍：
	前台系统实现（客户使用展示页面）功能介绍：

		网站首页，故事动态，股票信息，股票分类，购物车，会员登录，
		会员中心，管理员登录，管理员中心等核心功能展示

	后台系统实现（公司内部用户管理网站数据）功能介绍
		后台系统登录功能，主菜单页面，退出登录功能，修改用户的用户名和密码，
		修改当前用户的密码和资料，订单信息管理功能

详情看系统设计与论文实现文档。
